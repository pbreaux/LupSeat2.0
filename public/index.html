LupSeat

Assigns seats to students in a smart and automatic away.

LupSeat is a seat assignment generator specialized for applications for exam scenarios. It provides an quick and easy way for instructors to create a new seat layout.


LupSeat can be used from the command line but also comes packaged with a GUI. This program is cross-compatible, with distributions ready for both Mac and Windows.


Link to repository and releases.

The output of LupSeat is simple and graphical. It is ready to be printed out of the box, supporting most standard paper sizes.


The seats are spaced out as far as possible, minimizing the number of students in each chunk of seats using an algorithmic approach.
